#  LocoMed

LocoMed is a smart web application designed to help users quickly find nearby pharmacies that have the required medicines in stock.

##  Features

-  Login Page
-  Detects user location
-  Search for medicines easily
-  Shows nearby real pharmacies
-  Displays medicine availability from custom dataset
-  Google Maps integration for navigation
-  Distance calculation to nearby pharmacies
-  Fast and simple user interface

##  Tech Stack

- HTML
- CSS
- JavaScript
- JSON (for data storage)
- Google Maps Embed API

##  How It Works
1. User logins through the login page
2. User allows location access
3. Enters the required medicine
4. System checks the database
5. Displays pharmacies where medicine is available
6. User can view location on Google Maps

##  Future Scope

- Real-time pharmacy inventory integration
- Online medicine booking
- Prescription upload feature
- Emergency mode for 24/7 pharmacies
